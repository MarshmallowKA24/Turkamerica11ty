Preguntas Frecuentes

¿Es gratuita la aplicación?

Sí, la aplicación es completamente gratuita con todas las funciones desbloqueadas.

Sí, necesitas una cuenta de TurkAmerica para sincronizar tu progreso entre dispositivos.

Sí, puedes estudiar offline. Los cambios se sincronizarán cuando tengas conexión.

¿Puedo importar mis propias tarjetas?

Sí, puedes crear tus propias tarjetas o importarlas desde archivos CSV.
