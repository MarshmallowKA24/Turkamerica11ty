Recursos de Aprendizaje de Turco - TurkAmerica

Buscar Temas

Bildirme Ekleri (-DIr) (Presente)

Avanzado

Expresar hechos, verdades generales y suposiciones (probablemente).

Bildirme Ekleri (-DIydı, -mIştI) (Pasado)

Avanzado

Tiempos compuestos: "iba a...", "había...", "solía...".

Zarf-Fiiller (-IncAyA kadar, -dIkçA)

Avanzado

Expresar "hasta que..." y "a medida que...".

Sıfat-Fiiller (Ortaçlar) - Parte 1

Avanzado

Participios adjetivales (que hace): -An, -AsI, -mAz, -Ar.

Sıfat-Fiiller (Ortaçlar) - Parte 2

Avanzado

Participios adjetivales (que hizo/hará): -DIk, -AcAk, -mIş.

Edilgen Çatı (Voz Pasiva)

Avanzado

Expresar acciones donde el sujeto recibe la acción ("ser hecho").

Ettirgen Çatı (Causativa) - Parte 1

Avanzado

Hacer que alguien haga algo (Sufijos: -dIr, -t).

Ettirgen Çatı (Causativa) - Parte 2

Avanzado

Hacer que alguien haga algo (Sufijos: -Ir, -Ar, -Er, -It).

Oldurgan Çatı (Factitiva)

Avanzado

Convertir verbos intransitivos en transitivos (ej. dormir -> hacer dormir).

Zarf-Fiiller (-mAdAn önce, -dIktAn sonra)

Avanzado

Secuenciación de tiempo: "antes de..." y "después de...".

Zarf-Fiiller (Modo/Secuencia - Repaso)

Intermedio

Repaso de B1: -(y)Ip (y), -(y)A...-(y)A (repetición), -(y)ArAk (modo).

Bağlaçlar (Conjunciones: ki, diye, vb.)

Avanzado

Uso de conjunciones clave: ki (que), diye (diciendo/para), bu yüzden (por eso).

Dolaylı Anlatım (Estilo Indirecto) - Afirmativas

Avanzado

Reportar lo que alguien dijo ("Dijo que..."): -DIğI, -AcAğI.

Dolaylı Anlatım (Estilo Indirecto) - Preguntas

Avanzado

Reportar preguntas ("Preguntó si..."): -Ip -mAdIğI.

Dolaylı Anlatım (Estilo Indirecto) - Órdenes

Intermedio

Reportar órdenes ("Dijo que hiciera..."): -mAsI (Repaso B1).

Zarf-Fiiller (Repaso: -Ir -mAz, -(y)AlI)

Intermedio

Repaso de B1: "tan pronto como..." y "desde que...".

İsim Cümleleri (Cláusulas: -mAsI, -DIğI)

Avanzado

Cláusulas sustantivas: "El hecho de que..." y "Que (alguien) haga...".

Bağlaçlar (Repaso: hem... hem, ne... ne)

Intermedio

Repaso de conjunciones dobles: "tanto... como...", "ni... ni...", "o... o...".
